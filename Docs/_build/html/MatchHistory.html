MatchHistory.getDF_MH(league: str, season: str, download: bool = False) → DataFrame[fuente]

Esta función extrae los datos del historial de partidos de una liga y una temporada determinadas, y los devuelve como un marco de datos de pandas, con la opción de descargarlo como un archivo CSV.

Args:

league (str): una cadena que representa el nombre de la liga para la que queremos obtener los

datos del historial de partidos.

season (str): El parámetro de temporada es una cadena que representa el año o rango de años para

los que el usuario desea obtener los datos del historial de partidos. Por ejemplo, «2021» o «2019-2020».

download (bool): Un parámetro booleano que indica si descargar el DataFrame resultante como un

archivo CSV o no. Si se establece en True, la función guardará el DataFrame como un archivo CSV en la carpeta «Descargar» con el nombre «Picks&Bans-{liga}_{temporada}.csv». Si se establece en Falso, el. Defaults to False

Returns:

un marco de datos de pandas que contiene datos del historial de partidos para una liga y una

temporada determinadas. Si el parámetro de descarga se establece en Verdadero, la función también guardará los datos como un archivo CSV en una carpeta «Descargar». Si no hay datos disponibles para la liga y la temporada dadas, la función imprimirá «No hay datos» y devolverá Ninguno.

La función toma datos JSON, split, semana y un valor booleano y devuelve un Pandas DataFrame que contiene datos del historial de partidos para los jugadores de League of Legends.

Args:

json_data (dict): un diccionario que contiene los datos de la coincidencia en formato JSON split (str): Una cadena que representa la división del historial de partidos (por ejemplo, «Split

de verano»).

week (str): Un parámetro de cadena que representa la semana del historial de partidos. V4 (bool): Un parámetro booleano que indica si los datos están en formato V4 o no. Defaults to

False

Returns:

un DataFrame de pandas que contiene datos del historial de coincidencias para una división y una

semana determinadas, extraídos de una entrada JSON. Los datos específicos incluidos en el DataFrame dependen del valor del parámetro V4.
